PORTFOLIO MOULKA LAKHAL - MODE D'EMPLOI

Fichiers inclus :
- index.html : page principale du portfolio
- style.css : design du site
- script.js : animations et menu mobile
- lettre-motivation.html : lettre de motivation projective
- cv-projectif.html : CV projectif
- assets/ : dossier pour ajouter tes preuves, photos et PDF

Pour ouvrir le site :
1. Dézippe le dossier.
2. Double-clique sur index.html.

À personnaliser :
- Ajouter ton vrai numéro de téléphone si besoin.
- Remplacer les SAÉ par les noms exacts de tes SAÉ si tu les as.
- Ajouter tes PDF anonymisés dans assets/preuves/ avec ces noms :
  skills-office.pdf
  les-pleiades.pdf
  phencia.pdf

Important : anonymise les documents avant de les déposer dans le site.
